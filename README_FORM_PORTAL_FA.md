# Travel Market Forms Portal

این نسخه برای ساب‌دامین فرم‌ها آماده شده است.

- صفحه اصلی شامل کارت‌های فرم و کارت‌های راهنما است.
- فرم‌های فعال اسپانیا در پوشه `programs` حفظ شده‌اند.
- راهنماهای اسپانیا و پرتغال در پوشه `guides` قرار دارند.
- اتصال فرم‌ها به اسکریپت فعلی حفظ شده است.

برای انتشار، محتویات این پوشه را روی ریپوی متصل به Vercel آپلود کنید.
